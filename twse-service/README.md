# 台灣股市即時股價服務
## 目的
- 提供 user 需要的股票資訊，像是成交價、開盤價、揭示買賣價等
- 與 user 端紀錄的現有資產比對計算，提供即時損益評估

## API URL
1. [台灣證券交易所](https://mis.twse.com.tw/stock/api/getStockInfo.jsp)
  - 參數:
    - `ex_ch`: 帶股票代號，可用 `|` 傳多個股票代號；上市股票代號格式為 `tse_{}.tw`，上櫃股票代號格式為 `otc_{}.tw`
    - `json`: 
    - `delay`
    - `_`
